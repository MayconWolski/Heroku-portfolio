Sobre Mim

Um pouco sobre mim, sou um desenvolvedor Python e programador Web de Barueri, em São Paulo, e trabalho com desenvolvimento de automações de tarefas com Python e criação de páginas Web.

Faço projetos na área de programação desde 2020. Iniciei com o aprendizado de Python no Blender (desenvolvendo games) e hoje conheço várias estruturas diferentes e bibliotecas python e também bastante conhecimentos de estruturas web como: HTML, HTML5, CSS, CSS3, Javascript. Tenho um bom conhecimento com o Bootstrap e Programação Orientada a Objetos. Até o momento, criei sistemas nos quais sou muito grato, pois tive que encarar inúmeros desafios e resolver grandes problemas que me ajudaram a ser o profissional que sou hoje.
